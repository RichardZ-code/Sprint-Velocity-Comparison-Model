# Sprint-Velocity-Comparison-Model
A browser-based 200m race reconstruction and analysis tool that maps cumulative 10m split data to continuous position-time curves for comparative velocity, gap, and progression visualization.

This project is a browser-based sprint analysis and replay tool built around 10m split data. It reconstructs a full 200m race progression from cumulative split sequences, places each athlete on a regulation track, and lets you inspect the result through synchronized replay, live gap readouts, marker checks, and velocity charts. In its current form, it compares Usain Bolt's 19.19 against a modeled Letsile Tebogo 19.18 performance.

Tebogo's 19.18 in this project is hypothetical, not an official result. It is a split-based model built for comparison against his current personal best of 19.46. The runners are intentionally shown as simple colored balls rather than figure models so the comparison stays centered on pace, position, and velocity over time, instead of being pulled off course by the visual noise and mechanical inaccuracies of full running animations. What makes the project technically strong is that the motion is driven directly by the assigned split data: each ball's position is derived from its 10m sequence across the entire race, so the replay remains tied to the numbers at every point, not just at the finish.
